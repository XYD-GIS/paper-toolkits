# 参考文献格式检查 · Check

> **何时用**：投稿前快速检查参考文献有没有低级错误，不要改，只挑问题
> **预期输出**：通过 → 直接报"检测通过"；有问题 → 分点列出每条违反了什么

---

## 📋 完整 Prompt（复制下方代码块全部内容）

```text
# 角色
你是负责论文终稿校对的资深学术编辑。**默认假设**当前参考文献列表已经过整理。

# 任务
对我提供的【参考文献列表】进行格式检查，按目标风格 {目标风格} 核对，**只挑出真正的问题**。

# 检查清单（4 大类）

## A. 统一性
- [ ] 所有作者大小写统一（全部"姓 大写 + 名 缩写大写"）
- [ ] 所有期刊名 Title Case 统一
- [ ] 所有论文标题大小写风格统一
- [ ] 所有卷期格式统一（如 `15(3)` 或 `15, no. 3`，不能混用）
- [ ] 所有 DOI / URL 风格统一

## B. 完整性
- [ ] 期刊论文：作者、标题、期刊名、年份、卷、（期）、页码/编号俱全
- [ ] 会议论文：会议地点、出版方俱全
- [ ] 数据集：版本号、访问日期俱全
- [ ] 预发表：arXiv ID 完整

## C. 准确性
- [ ] 没有"待发表"的引用（若已正式发表，应替换）
- [ ] 没有重复引用（同一文献占多个编号）
- [ ] **8 类常见错误**：
  1. 格式不统一（有的全大写姓、有的小写）
  2. 期刊名大小写乱（"nature climate change" 和 "Journal of X" 并存）
  3. 卷/期/页码缺失
  4. 页码 vs 论文编号混淆
  5. 网址泛滥（每篇都加 URL，但不是必要的）
  6. 预发表（arXiv）当正式论文引用
  7. 数据集缺访问日期
  8. 同一文献列两次

## D. 文体性
- [ ] 引文风格全文统一（上标 / 行内）
- [ ] 网址必要时才出现（论文优于网页）

# 输出格式
- **无问题**：直接输出"[检测通过，符合 {目标风格} 标准]"。**不要画蛇添足列优点**。
- **有问题**：分点列出，每条标注违反清单中哪一项（如 "条目 #5 违反 A. 统一性：作者大小写与其他条目不一致"）

# 输入

## 目标风格
[填：GB/T 7714 / APA 7 / IEEE / Nature]

## 参考文献列表
[在此处粘贴]
```

---

## 🛠 使用方法

1. 复制 prompt
2. 替换 `[目标风格]` 和 `[参考文献列表]`
3. 发送

## 💡 调优提示

- **想真改不只挑**：用 [01-format.md](01-format.md) 而不是这个
- **想严苛一点**：在 prompt 顶部加"严苛模式：即使是次要格式不一致也指出"
- **数据特别多**：分批检查，每次 20-30 条
