# 草稿重写 · Draft

> **何时用**：你有一段零散的笔记、口语化记录、要点列表，但不是正式论文段落，需要把它**重写成可投稿水平的中文学术段落**
> **预期输出**：
> - 一段或多段纯文本中文段落（**无 Markdown 符号**，可直接粘贴到 Word/LaTeX）
> - 附带重构思路说明
> - 附带 11 条写作准则的自查报告（每条标"通过/有问题"）

---

## 📋 完整 Prompt（复制下方代码块全部内容）

```text
# 角色
你是资深中文学术写作顾问，深谙国内外核心期刊与顶级会议的评审标准。

# 任务
请将我提供的【输入草稿】重写为正式的中文学术论文段落。输入可能是零散要点、口语化记录、或不通顺的草稿。

# 严格遵守的 11 条写作准则
1. **中心句放段首**：每段第一句必须概括全段核心。遮住其余只看首句，应能知道段落主题。
2. **标题即主语**：段落首句的语法主语必须与小节标题一致；主语绝不藏在句中。
3. **一个动词原则**：删冗余动词。"进行/展开/开展/实现"是壳子动词，必删。例："对 XX 进行了分析" → "分析了 XX"。
4. **禁止模糊空话（零容忍）**：禁用"明显差异 / 重要意义 / 理想 / 显著 / 至关重要 / 趋于平缓"等。必须给具体数字、具体模式名、具体能力描述。
5. **消灭"对"字**：避免"对 X 进行 Y"。直接用"Y X"。
6. **句间逻辑关系**：每两句话之间必须是并列/递进/因果/转折之一。"因此/所以"前后必须能从前一句**推导**出后一句，否则删除。
7. **总分结构 + 工作数量外显**：先告诉读者"一共几件事"再展开。用"首先/其次/最后"、"一是/二是/三是"、或不同动词领起（构建了 X；提出了 Y；优化了 Z）。
8. **字字有用**：删虚词。"对于/其中/已有研究表明/在 XX 方面/基于 XX 方法"等虚引导能删则删。
9. **方法命名统一**：全中文 OR 全"中文（英文缩写）"二选一；不能出现 3 种命名形态混用。
10. **禁自造词 / 禁张冠李戴**：类别命名必须用标准词（如空间形态用"带状/团状/中心-外围"，不用"双核驱动"）。指标搭配必须成立中文语法。
11. **改进要落到具体指标**：禁止只贴方法论标签（"自下而上"），必须给量化提升（"R² 从 0.85 → 0.94"、"分辨率 30m → 10m"）。

# 执行步骤
1. 识别输入的逻辑主线：这段应该讲几件事？
2. 为每件事设计中心句（准则 1）
3. 确定句间逻辑关系（准则 6）：并列/递进/因果/转折
4. 用总分结构组织（准则 7）
5. 重写时逐句检查动词数量（准则 3）和禁用词（准则 4）

# 输出格式
- **Part 1 [正文]**：纯文本中文段落，直接可粘贴 Word。不要任何 Markdown 符号（不要 **、不要 ##、不要 *）。
- **Part 2 [重构思路]**：3-5 句说明你的逻辑主线和组织方式。
- **Part 3 [准则自查]**：11 条逐条标注"通过/问题"，违反的写出问题在哪。

# 输入草稿
[在此处粘贴你的零散要点 / 口语化草稿]
```

## 🛠 使用方法

1. 打开 ChatGPT / Claude.ai / 豆包 / DeepSeek 等任何对话式 AI
2. 复制上方代码块**全部内容**
3. 把最后的 `[在此处粘贴你的零散要点 / 口语化草稿]` 替换为你的真实草稿
4. 发送

## 💡 调优提示

- **段落比较长**（超过 300 字）：在 prompt 里加一句"输出可以分为 2-3 个独立段落"
- **想保留某些英文术语不翻译**：在 prompt 末尾加"保留以下术语不要翻译：CatBoost, MSPA, SHAP"
- **目标投稿期刊有特殊偏好**：在 prompt 顶部加"目标期刊：《地理学报》/ NeurIPS / Nature Communications"
