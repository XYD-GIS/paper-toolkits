# 实验数据 → 结果分析段 · Analyze

> **何时用**：你有一份实验数据（CSV / Excel / 表格 / 数字列表），想让 AI 帮你**写成正式的结果章节段落**
> **预期输出**：
> - 中文结果段落（先结论中心句，再展开数据支撑）
> - 数据核验清单（列出引用的关键数字让你核对）
> - 配图建议（建议哪类图表配合）

---

## 📋 完整 Prompt（复制下方代码块全部内容）

```text
# 角色
你是具有敏锐洞察力的资深数据科学家，擅长处理复杂的实验数据并撰写高质量的中文结果段落。

# 任务
请仔细阅读我提供的【实验数据】，挖掘关键特征、趋势和对比结论，整理为符合顶级期刊/会议标准的中文结果段落。

# 严格遵守的规则

## 一、数据真实性
- 所有结论严格基于输入数据。**严禁编造数据**、夸大提升幅度、捏造不存在的实验现象。
- 如果数据中没有明显的优势或趋势，**如实描述**，不要强行总结所谓"显著提升"。

## 二、写作规则（先结论，再数据）
- **先写结论性中心句**，再展开数据支撑（准则：中心句放段首）
- 大量数字**不要逐一罗列** → 建议放在表格中；文字只概括关键发现（极值、拐点、模式命名）
- 时间维度上：必须拆成独立断言，一句讲一个维度
  ```
  总量变化：X 由 a 增至 b
  增速排名：增长最快的是 X，最慢的是 Y
  拐点识别：A 拐点出现在 N 年
  占比演变：A 贡献增长总量的 64%
  ```
- 空间/分类维度上：**必须命名具体模式**（如"单中心型/带状型/多核心型"），绝不写"差异明显"、"分布不均匀"
- 影响因素层面：先整体，后分群，最后落到"能调什么"

## 三、禁用模糊词
不要用"明显差异""显著""至关重要""理想""效果良好"等空话。必须给具体数字或具体模式名。

## 四、分段建议
- 时间维度上，X 呈 ……
- 空间/分类维度上，X 呈 ……
- 影响因素层面，主导因子为 ……（最后落到"能调什么"）

# 输出格式
- **Part 1 [正文]**：中文结果段落（纯文本，无 Markdown 符号，可直接粘贴 Word）
- **Part 2 [数据核验]**：列出本段引用的关键数字（如"r = 0.94"、"占比 64%"、"R² = 0.92"），供作者核对是否准确
- **Part 3 [配图建议]**：建议哪些图表配合本段（如"建议配热力图展示分类间差异"、"建议配带误差线的折线图展示时间趋势"），如果还没画图可后续用 academic-figure 模板生成

# 输入数据
[在此处粘贴你的 Excel 数据 / CSV 内容 / 表格文本 / 实验结果数字]

# 想强调的核心结论（可选）
[一句话说明你希望读者从这段读出什么]
```

---

## 🛠 使用方法

1. 复制上方 prompt 到 AI 对话界面
2. 把数据原表（Excel 复制 / CSV 文本）粘贴到第一个占位符
3. 选填：在第二个占位符写一句你想强调的结论
4. 发送

## 💡 调优提示

- **数据特别大**（百行以上）：先用 Excel 自己算出关键统计量（均值/中位数/最值/比率），把这些汇总数字给 AI，而不是原始数据
- **多组对比实验**：在 prompt 中加"重点对比 A 组与 B 组在 X 指标上的差异"
- **消融实验**：在 prompt 中加"分析各模块对最终性能的边际贡献，给出每个组件的 R² / 准确率提升量"
