# 常见错误 + 格式核对清单

---

# 一、8 类最常见错误

参考文献格式混乱会让评审直接怀疑论文的专业程度，是性价比最低的失分点。

| # | 错误类型 | 表现 | 后果 |
|---|---|---|---|
| 1 | 格式不统一 | 同一列表里有的全大写姓、有的小写；有的卷期全、有的缺；有的带 DOI、有的不带 | 评委一眼看出业余 |
| 2 | 期刊名大小写乱 | "nature climate change" 和 "Journal of Computational Physics" 并存 | 不专业 |
| 3 | 卷/期/页码缺失 | 只有 "Nature, 2024" 没卷期页码 | 无法定位文献 |
| 4 | 页码 vs 论文编号混淆 | 部分期刊用文章编号（如 105）而非页码，作者却两种都写 | 错误引用 |
| 5 | 网址泛滥 | 每篇都加 URL，但 URL 是非永久链接 | 既冗余又不稳定 |
| 6 | 预发表（arXiv）当正式论文 | "Smith et al., 2024" 但实际只是 arXiv preprint | 引用未发表 |
| 7 | 数据集缺访问日期 | 数据集引用只有 DOI，没有 "Accessed: 2026-04-26" | 不可追溯 |
| 8 | 同一文献列两次 | 同一篇文章占了两个编号 | 评委直接扣分 |

# 二、提交前必查清单（4 大类）

### A. 统一性

- [ ] 所有作者大小写统一（全部 "姓 大写 + 名 缩写大写"）
- [ ] 所有期刊名 Title Case 统一
- [ ] 所有论文标题大小写风格统一（首字母大写 或 全 Sentence case）
- [ ] 所有卷期格式统一（如 `15(3)` 或 `15, no. 3`）
- [ ] 所有 DOI / URL 风格统一

### B. 完整性

- [ ] 期刊论文：作者、标题、期刊名、年份、卷、（期）、页码/编号俱全
- [ ] 会议论文：会议地点、出版方俱全
- [ ] 数据集：版本号、访问日期俱全
- [ ] 预发表：arXiv ID 完整

### C. 准确性

- [ ] DOI 全部经过 doi.org 验证
- [ ] 没有 "待发表" 的引用（已被替代为正式发表版）
- [ ] 没有重复引用（同一文献占多个编号）
- [ ] 出现顺序与正文一致

### D. 文体性

- [ ] 所有英文缩写首次出现有中文/英文全称
- [ ] 引文风格全文统一（上标 / 行内）
- [ ] 网址必要时才出现（论文优于网页）
