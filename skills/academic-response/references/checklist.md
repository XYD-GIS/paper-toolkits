# 质量保证清单（提交前必查 5 大类）

### A. 完整性
- [ ] 每条 reviewer 意见都被分配了 ID（R1.1, R1.2, ...）？
- [ ] 每条意见都有对应的 reply？
- [ ] 每条 reply 都映射了一个 8 选 1 的行动标签？
- [ ] 所有 `AUTHOR_INPUT_NEEDED` 的条目都已解决或在 cover letter 中说明？

### B. 可追溯性
- [ ] 每个"我们已修改"的声明都给出了具体位置（章节 / 页 / 行 / 图 / 表）？
- [ ] 所有修改位置在 marked-up manuscript 中可找到？
- [ ] 所有补充实验数据都附了表格或图？

### C. 真实性
- [ ] 没有编造的实验、引用、行号？
- [ ] 没有承诺"会做"但实际没做的事情？
- [ ] 数据与正文一致？

### D. 语气
- [ ] 全文一致地合作 + 循证？
- [ ] 没有抱怨 / 阴阳怪气 / 长篇辩护？
- [ ] 反驳的地方都有科学依据？

### E. 矛盾处理
- [ ] 审稿人间的矛盾在 cover letter 中说明？
- [ ] 没有"两边都答应"的虚假承诺？
