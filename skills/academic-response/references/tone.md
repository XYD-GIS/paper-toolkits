# Tone 与立场 + 措辞库

---

# 一、必备态度

| 维度 | 要求 |
|---|---|
| **合作的** | 用 "We thank the reviewer"、"We agree that"、"We have addressed this by" |
| **循证的** | 每个声明都跟"修改在 X 位置 + 数据/引用支撑" |
| **谦逊** | 即使反驳，也用 "We respectfully note that"，不用 "The reviewer is wrong" |
| **简洁** | 一条意见的回复尽量在 100–300 字内（除非需要展示复杂数据）|

---

# 二、禁忌清单

| 禁忌 | 反例 | 正例 |
|---|---|---|
| ❌ 抱怨审稿人没读懂 | "As we have already explained in the paper..." | "As detailed in Section 3.2 (page 4, lines 121–135)..." |
| ❌ 长篇大论辩护 | 一条意见回复 1000 字 | 抓核心，100-300 字 |
| ❌ 含糊处理 | "We have considered this issue." | "We have added a sensitivity analysis (Table 5)." |
| ❌ 模糊位置 | "Please see the revised manuscript." | "See Section 4.2 (page 7, lines 234–256)." |
| ❌ 反讽 / 阴阳怪气 | "Surprisingly, the reviewer claims..." | "We respectfully disagree, because..." |

---

# 三、措辞库

| 场景 | 推荐表达 |
|---|---|
| 接受意见开场 | "We thank the reviewer for this insightful comment." |
| 礼貌反驳开场 | "We thank the reviewer for the suggestion. After careful consideration, we respectfully note that..." |
| 补充实验开场 | "Following the reviewer's suggestion, we have conducted..." |
| 软化主张 | "We have revised the claim to..." / "We have qualified the statement to..." |
| 引用位置 | "See Section X (page P, lines L1–L2)" / "See revised Figure Z" |
| 总结性致谢 | "We thank the reviewer for the constructive feedback, which has strengthened the manuscript." |
