# 格式规范速查

| 项目 | 要求 |
|------|------|
| 英文和数字字体 | Times New Roman |
| 英文标点后 | 必须加空格 |
| 英文缩写首次出现 | 中文全称（英文缩写），如"主成分分析（PCA）" |
| 标题编号与文字 | 之间必须有空格："2.3 数据预处理" |
| 公式后 | "式中，X 代表……" 逐一解释每个变量 |
| 图表 | 正文中必须引用（"见表 X"、"如图 Y 所示"）|
| 排放系数等常数 | 放表格或附录，不放方法正文 |
| 行距 | 用固定磅值调成接近单倍行距 |
| 他人的图 | 绝不直接复制，必须自己重绘或获得书面授权 |
